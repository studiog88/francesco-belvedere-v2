# Where your Cursor rules live

The **full rules** Cursor loads are in:

**[`.cursor/rules/portfolio.mdc`](.cursor/rules/portfolio.mdc)**

Edit that file when you want to change agent behavior. There is **no** automatic sync between this stub and the `.mdc` file—this file exists so you have an obvious entry point from the repo root.

**Finder:** Folders starting with `.` are hidden by default. Press **Cmd + Shift + .** to show them, or use Cursor’s file sidebar (the `.cursor` folder is normal project files on disk—not stored only inside the Cursor app).
